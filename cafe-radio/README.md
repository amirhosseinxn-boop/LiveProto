# کافه رادیو - منو

یک وب‌سایت سبک و سریع برای نمایش منوی کافه (RTL، تم سفید/قرمز).

## اجرا
- روش ۱: فایل `index.html` را در مرورگر باز کنید.
- روش ۲ (پیشنهاد برای QR): یک سرور محلی اجرا کنید:

```bash
cd /workspace/cafe-radio
python3 -m http.server 8080
```

سپس آدرس `http://IP-شما:8080` را به QR تبدیل کنید.

## ویرایش منو
- آیتم‌ها در `menu.json` هستند.
- قیمت را در `price_rial` به ریال وارد کنید (مثلاً 220000 ریال = 22000 تومان).
- عکس هر آیتم را در `assets/img/` قرار دهید و مسیرش را در `image` بنویسید.
- دسته‌بندی با `category` مشخص می‌شود (مثلاً "نوشیدنی های گرم").

نمونه آیتم:
```json
{
  "id": 8,
  "title": "کافه گلاسه",
  "en": "Cafe Glace",
  "category": "نوشیدنی های سرد",
  "price_rial": 320000,
  "image": "assets/img/cafe-glace.jpg"
}
```
